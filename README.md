# MegARM
Module to in-place ATmega328P upgrade. GRBL WoodPecker board targeted particularly.

Based on 5V Cortex-M0+ 48MHz [Microchip SAM C2x family](https://www.microchip.com/wwwproducts/en/ATSAMC20E18A).
